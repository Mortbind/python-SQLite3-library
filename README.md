SQLite3 python库项目

python版本：Python 3.10+

⚠️ 注意：当前版本 WHERE 子句直接拼接用户输入，存在 SQL 注入风险，仅限本地学习使用。

此项目是我学习过程SQLite3 python库创建来实践的，因此此项目并不是一个很完善的项目，有很多BUG、以及一些错误处理没有等缺点，因此只建议看看玩玩，不建议使用。

主要功能：
主要包括了SQLite3 常用命令
1.CREATE
2.SELECT
3.UPDATE
4.DELETE
5.INSERT

额外功能：
1.SNOW_INFO：（参数：表名）这个方法用了显示数据实现原理就是"SELECT * FROM 表名"，然后获取整个数据表内容然后打印到终端。
2.ROLLBACK：用于回滚数据，因为我的修改数据表的指令最终都有提交，因此我设计这个方法放置数据误操作。

下面是主要功能的一些讲解：
1.CREATE：（参数：表名）
开始会让用户输入想要创建字段的数量，
之后就是一次字段的名称、类型、以及约束
类型：INTEGER，TEXT，REAL，BLOB"
约束：PRIMARY KEY，NOT NULL

2.SELECT：（参数：表名）
开始会让用户输入想要显示的字段，
之后就是让用户是否想输入WHERE表达式
是：就输入表达式的数量和表达式的内容
否：就不加入WHERE这个参数。
WHERE表达式：比如我想要 身高 > 180cm 的学生 就这样写 height > 180(180具体看表中默认用的是m还是cm)，
并且拼接用or或and

3.UPDATE：（参数：表名）
开始让用户选择要修改的字段，然后在输入想要用户修改字段的值，
然后同样让用户输入WHERE表达式，其执行的逻辑与SELECT中的一样。

4.DELETE：（参数：表名）
与SELECT一样只用输入WHERE表达式。

5.INSERT：（参数：表名）
开始让用户输入各个字段的数据，然后插入指定数据表。
